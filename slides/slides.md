The deck is generated by `build_deck.js` (PptxGenJS). Outputs:
`TS-FM-Short-Story.pptx` and `TS-FM-Short-Story.pdf` (18 slides, 16:9 wide).
Speaker notes are embedded in the `.pptx`; the full narration is in
`../video/video_script.md`.

To rebuild: `npm install && node build_deck.js`
